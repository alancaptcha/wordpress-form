== Description ==

ALAN Captcha is a privacy-first, GDPR-compliant anti-bot solution that protects WordPress website forms, logins, and more from spam, abuse, and attacks.

It is the ultimate captcha solution to safeguard your website and protect user privacy by utilizing unique crypto puzzles based on a proof-of-work mechanism.

This user-friendly, privacy-preserving reCAPTCHA alternative is easy to install—no technical skills required.
Start protecting logins, forms, and more in minutes. Fully GDPR & WACC compliant!

== How ALAN Captcha Works ==

ALAN Captcha distinguishes between humans and bots using a challenge-response test in the form of unique crypto puzzles, solved by the user’s device. No user action is required.

As a proof-of-work-based anti-bot solution, it is low-friction, cookieless, and provides maximum protection while ensuring compliance with EU data protection regulations.

== Installation ==

Super quick and easy to get started with ALAN Captcha.

1. Register at www.alancaptcha.com and generate a Site Key and API Key.
1. Download and install the plugin.
1. Activate the plugin through the Plugins menu in WordPress.
1. Enter the Site Key and API Key from your ALAN Captcha account in the plugin settings.
1. Select the forms you want ALAN Captcha to protect and click Save.

Alternatively, install the plugin manually:

1. Download the .zip file from the Releases section.
1. Upload it to the /wp-content/plugins/ directory.
1. Activate the plugin through the Plugins menu in WordPress.
1. Enter your Site Key and API Key in the General Settings tab (Settings > ALAN Captcha).
1. Review the settings and enable the desired integrations.
